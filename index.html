  • 写在wowaka离去之后

    2019-04-08 /
    • { 随笔 }

    今天早上突然在微博上看到一张推特截图，ヒトリエ官方发的——wowaka的讣告。怎么可能呢，我记得wowaka是位年轻有为的P主，去年才出过新歌。以为又是那种我国互联网流行的“老顽童爷爷的扮演者今天去世了”之类的假新闻。

    然而看到之后心里总是不安——讣告是日文写的，格式公整且用词规范，实在想不出伪造这种信息的必要性。于是特意去推上确认。我之前没关注过P主相关的账号，搜了wowaka，看到他的最近一条po，4月1日发表：

    「令和きれいだー。」

    “令和，真美啊。”

    好像没什么问题呀？才更过推，果然是假新闻吧。稍稍安心了些。

    然后看到第一条评论，10分钟前发表。

    不对。

    「綺麗だから戻ってきてください。

    　まだ綺麗な世界を見せてください」

    “很美啊，所以请你回来吧，请再看看这个美丽的世界。”

    脑子先是空白了一下，然后就想不到任何说的了。过了好一会再回过神来。其实瞬间涌上了很多词语：天妒英才，英年早逝，事发突然……难以置信。

    想了很多，回忆了很多，他去年才出过新歌，还在niconico上自己弹唱过，当时wowaka自己还吐槽这歌太快了好难唱啊。想起初中时候单曲循环rolling girl & 里表lovers的日子，想到小缘在七、八年前时，作为一个5sing小有名气的歌手翻唱过他的unhappy refrain，而那时她还没成为现在斗鱼的百万大主播。想到二周目小圆时，and I’m home那句「ここにいるよ」 带来的感动。

    这些碎片的回忆说起来没什么意义，毕竟只要是从doriko & ryo & 暴走P 那个时代过来的听众们，各人都有各人的经历。如果不是去年那首アンノウン　マザーグース，我可能也不会发现原来他也已经有了自己的乐队，也时常担任主唱参加各种演出。

    现在唯一有意义的事实就是：wowaka永远离开了。

    直至此时才发现，其实我向来很抗拒主动思考死亡的意义。或许是因为年纪尚轻，并未经历过与亲密之人的生离死别。文学或影视创作里的死亡，不论再怎么渲染，还是因为虚构而显得轻飘飘的——像浮在水上的碎石，你能隐约感受到它的重量，可还是会被水带来的视觉效果欺骗。

    可对死亡的感知是真实存在的。六年级时看《Claymore》，看到Teresa的首级被瞬间取下，之后甚至不愿意打开那一卷漫画；高中时半夜看《蓝宇》，看过胡军走进太平间的那场戏后，哭得眼睛肿，几天精神恍惚走不出来。上个月看火影漫画，看到鸣人的妈妈在死前对着面前的婴儿千叮咛万嘱咐，“你要好好吃饭尊敬师长多交朋友”，二十来页漫画翻过去，满桌子的眼泪。

    如此看来，比起生者，死亡对于当事人倒是件更轻松的人。眼前一片漆黑，了却生前身后名，从此一切都可以不管不顾，不再过问。可生者要承受的是永久的离别，明知绝不返航，却仍要向前。Teresa死去了，读者的眼泪在说，你死了，要留Clare一个人怎么办。蓝宇死了，观众想的也是陈捍东究竟要怎样做，才能从回忆脱身。鸣人从未见过自己的父母，生即为孤儿，身上带着诅咒一般的事物，他今后的生活会有多艰难也可想而知。

    wowaka的离开也一样。他与他的作品为无数的人制造过有价值的回忆，而他的离开就意味着这份宝贵回忆的制造者不复存在，而他今后也绝不可能继续创造更多「Vocaloidっぽい」的作品了。比起前者，可能后者才是更让人悲痛的。

    抛开死亡这个既定事实，这又印证了我一直以来的想法：不论你多么习以为常的事物都不会如你所愿地陪你永存下去，它一定会在某一天渐渐或突然地离开你。

    初次有这个想法，是IGICE解散的那天。时间：17年7月；离别指数：一个星期的眼泪/不论白天黑夜只要想起就会哽咽。你以为他们会在休赛期好好整顿，到了下个赛季，还是那么要好的6个人站在同一个舞台上，却在一天内因为可笑的原因各奔东西。

    第二次，是6324倒闭的那段时间。抽象一台还在的时侯，三混神天天被骂混吃等死，但一旦他们不在一起了，你却又开始怀念。我甚至怀疑是不是我喜欢的团体终究都会迎接这样熟悉的剧情：反目成仇，之后分崩离析，各谋出路。

    然后就是今天wowaka的突然离开。平时太过于习以为常，此时现实便显得更突兀。

    不过这个感悟倒是经常出现。高三那年，年关的某一天，在亲戚家吃饭，突然就想通了这一点：你所经历的一切都不再有再来一次的机会。就像高三学生永远想要逃离高三，但一旦离开却发现现在即使想要受那时的辛苦也没有机会了；你的哥哥姐姐只有唯一的一段大学经历，按部就班地走入社会，成为社会中的零件之一——每个人都一样。

    只有时间例外——不可知，不可逆，不可说。

  • 你真的想回到过去吗？

    2019-03-27 /
    • { 随笔 }

    今天买的火影护额引起我不少感触，它让我想起小时候经过门口小商店时内心复杂的感受：想看，想买，但是没有钱，就算有钱，若是被家长发现自己买了这些没用的东西，也会招致斥责。我一直都这么觉得：其实当未成年小孩子真没那么好。

    小学的时候陕4每天下午放学时间，会播一部陀螺战斗类的电视剧（其实只是卖陀螺的厂家拍的宣传用片，类似现在的软广，不过那时小孩子们都很吃这一套）。我那时特别喜欢。攒了10块钱（2010年左右）去买了个周边——一个黄色的项链，代表着雷电。项链的质感很不错。

    戴上后的第一天，我理所应当去学校炫耀。班里一个男生非要拽过去看，一拉一耸的就把链子拽断了。

    我当时觉得天都塌了，但面对同学也说不出什么。只能说，我这是十块钱买的，也不跟你加钱了，你就还我10块。

    催了一学期也没要来。我只能寄希望于学期末的家长会。那时小孩子心里的大人就是正义的，也不会考虑立场这回事。

    家长会结束后，义愤填膺地跑去那个男生的妈妈面前，说了这个事。在我脑海中反复上演了一学期的——他妈妈大发雷霆回家把他暴打一顿，再带着他来给我道歉——的戏码，没有发生。他妈妈反应淡淡的，说知道了。转过天，他没给我钱，给了我一条和原来那条不一样造型的项链，是另一个人物的。

    我心里并不满意这个结果，可也没什么办法。最后仍是不了了之。

    说了这么多，很杂乱，但并不是停留在表面上的“看到了火影回忆起了童年”。其实有很多更深层的东西。我有时候感觉我在情感方面很冷酷，人类社会中上到八十岁老人下到十几岁的小孩，大家都会不时回忆以前的生活，伴随发出想要回到过去的感慨。可我很少有这种经历。王姐cx总说想回到高中，而我和朱麒臻能在coco人手一杯茉莉绿茶从下午坐到晚上一起回忆初中，笑得坐不起身，晚上十点了还趴在吧台上不走。

    但我的怀念也就止步于此。小学意味着奥数的折磨与全方位的不自由的管制，而高中除了能够吞噬一切的自卑以外就是考试成绩，再无它物。高三时看到很多激励性质的语言，听的最多的无非那句“高考完的考生以为自己解放了，实际上他们刚刚离开了天堂”。我从那时就对这句话嗤之以鼻——即使高中有过一星半点的美好回忆，我也坚信，这些细碎的东西，永远无法与它带给我的痛苦相匹配。我那时就明白，即使我高中毕业，走入大学，步入社会，也绝不会存在那样的时刻——我怀念我的高中，想要再次投身到那个冰冷而强硬的大环境中去。

    所以说，长大真好。除了情感丰富之外什么也没有的日子，本质上仍是贫乏的。小学时的你不愿和所谓的朋友分别，高中时的你却已经不会再回首。随着长大，你的负担会越来越重，但也托了成长的福，你摆脱了很多束缚。 就算十年过去了，我还是很厌恶小学时路过门口小商店时那种源自内心的不自由感，不论其初衷是否合理。

    其实你没那么怀念过去。别脱口而出，别理所当然。

  • I sometimes wish I'd never been born at all

    2019-03-25 |
    • { 波西米亚狂想曲 }
    /
    • { 影评 }

    没去看过正儿八经的演唱会，没听说过Queen，一开始甚至没准备去看的。今天下定决心买了万达的杜比厅。

    不知道音乐和故事哪一个带给我的震撼更多，但所有的音乐都足以给人以令人头皮发麻的震撼。

    总自诩拥有足够的共情能力，一开始我觉得我也许能理解他的孤独。的确他身上有我能够理解的部分——乐队人员被拉到一处偏僻的农场录制专辑，我看到Freddie拿着一截短短的铅笔在灯光昏暗的房间里文思泉涌的样子，看到他眼中半是欣喜半是悲伤的眼泪。我也经历过写东西把自己写哭的经历，可以前我从未认真想过为什么有时候我在写东西的时候会本能地止不住眼泪。就在看到那个镜头的时候我才反应过来：因为你笔下的文字准确地反映了此时内心最真实的想法。

    这件事听起来不难——用文字记录内心，简单得仿佛是某个小出版社用来做征文宣传语的句子。不过我觉得这事更像GRE中issue的要求：表达出你的立场不难，难的是表达出你的程度——看似是同一样事物，却像八分熟的牛排与五分熟的牛排般截然不同。每个人都能同时感受到十种复杂深刻的情感，可往往只能察觉到其中三种，将两种在脑中雕琢成形，再将最主要的那一种用文字或声音表达出来。

    这也是为什么人们，包括我在内，都热爱有天赋的人——他们能用最准确最动人的话语to make themselves understood。我想这是我能理解Freddie的部分。

    再说一些动人的细节。

    之一，我没想到他们创造了那么多经典：we’ll rock you以及那个具有感染力的简单舞蹈是他们发明的，那首奥运会一定会被拿出来作为BGM的we’re the champions是他们的歌曲。

    之二，在看到Freddie离开Paul的那个雨夜，心里莫名地轻松。我以为这会是个关于堕落的故事，但没想到他最终还是被爱拯救了。有时候我常想，那些独特的人理应具有堕落的特权，如果他们想要更激烈地感受最真实最复杂的情感，就不能只是和普通人一样无病呻吟，作出深有感悟的样子。他们理应体验痛苦与离别，感受眩晕与跌落，让一切负面的感情渗入每一个毛孔，再经由他们异于常人的天赋，让最剧烈最深刻的情感从笔下涌出。内容关乎爱与被爱，关乎渴求与绝望，关乎追逐与挣扎。也许他们也正深陷泥潭，可还是想燃烧自己的力量，给人以力量与救赎。

    之三，Freddie宣布自己染上艾滋时，对回归后的伙伴们说“我不希望你们拿同情来我这里寻找存在感”。我想起蜉蝣——向死而生。我没想到他们也会因此眼泛泪花，也是这时我才意识到死亡的意义。死亡理应是endless silence，但对于Freddie，死亡更像是一则doomed announcement。做完了手头的事后，无声离开。

    “I’m just a poor boy I need no sympathy” /

    所以我在听到最后Live Aid那句”Mama just killed a man”时开始不住地流泪。我在想，当他在千万人仰望的舞台上， 声嘶力竭地唱出”I don’t wanna die I sometimes wish I’d never been born at all”时， 内心在想什么。

    为堕落的生活忏悔，对辜负的爱人道歉，向过去的辉煌谢幕。

    He deserved it.

    CQL特性

    No arbitrary WHERE clause

    No JOINS

    No arbitrary GROUP BY

    No arbitrary ORDER BY clauses

    创建 create

    创建名为animals的keyspace后，可以用desc keyspaces查看。

    之后创建monkeys的表。

    CREATE TABLE monkeys (

    ​ type text,

    ​ family text,

    ​ common_name text,

    ​ conservation_status text,

    ​ avg_size_in_grams int,

    ​ PRIMARY KEY ((type, family), common_name));

    );

    注意此处的primary key。此处为一个复合键。前一个(type, family)为partition key，后一个common_name为cluster key。

    composite partition key

    由多列组成，在cluster columns之前由一组额外的括号定义。

    既然cluster key决定了数据在一行中存储的顺序 那么partition key就帮助将数据分散在cluster中。

    设计表时，可将partition key视为将数据平均地传播到cluster中的工具。可将cluster key视为决定一个partition中数据顺序的事物。

    *cluster key是可选的。

    插入 insert

    当插入数据时，主键是必需的（mandatory）。

    命令语句同MySQL数据库。

    之后用select语句查看数据，红色属性为partition key，蓝绿色（cyan）属性为cluster key。

    一致性等级 consistency level

    此属性可以配置。

    Cassandra作为NoSQL数据库在CAP原理上选择了AP，即availability和partition tolerance（未选择C which means consistency）。

    输入 consistency cql中出现表述：

    cqlsh:animals> consistency
    Current consistency level is ONE.

    解释：一致性等级为1，表示只要一个node成功插入了这个statement，则认为插入成功。在每一个节点上，该insert statement首先被写入commit log，之后再被插入memtable（如一个write back cache）。这个memtable会被周期性或是在达到某个

    bu’guo阈值时被冲入disk中。这个flushing的过程将memtable/cache转化为一个SSTable。

    nodetool flush animals
