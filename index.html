  • 探索生命中的荒诞与空白

    2019-05-07 /
    • { 书评 }

    What We Talked About When We Talked About Love – Raymond Carver

    卡佛为什么能写出“从水里抬起又落下去的胳膊，是在挥别好时光和招呼坏时光的到来…”这样的句子，对一具尸体的胳膊做如此解读，那种丧气的宿命感着实让我沉溺其中，无法自拔。

    还有“我母亲和别人乱搞时，我正在失业。我的孩子们疯了，我老婆疯了，她也正在和别人乱搞。那家伙是她在戒酒协会认识的，一个失业的航天工程师。他也疯了。”寥寥数语就捕捉住了时代的精神，生活的本质（本人的过度解读），不可思议，令人赞叹。（也许这是编辑的功劳….）

    我曾经认为能够被摘录好词好句的作家都是傻逼，卡佛显然不在此列，那样的文字无法给人启示，“没有营养”，找不到金句，得不出结论。它直白又晦涩，如同生活本身，是一团迷雾，但它让人平静，也许生活的本体才是焦虑的解药。

    废物作家写废物，庸常生活的细枝末节也能有文学上的意味，但底层故事终究不是卖给底层看的，这也是卡佛被一些人诟病的点，“中产喜爱的作家”。但这不是卡佛能决定的，作为一个作家，卡佛足够真诚。

    早年父亲是电工，他不看卡佛，他只是活着，并不觉得生活有什么痛苦，他只会在家里人问他要钱时唉声叹气，闭嘴，不要说了，然后就没了。“就当无事发生”我妈说。

    多么正常，生活常态，写成文字却感到悲哀。可能是因为对人生还有所期待，或者是矫情。

    那这是我自己的问题了。

    说实话我对外国文学的了解除了来自中学语文那点粗浅的知识以外，就是刘书宇了。初次在豆瓣上看到他的时候，立刻被消极厌世的文字吸引。我没想到他是学电气工程的，但知道了后，又觉得他像极了学电气工程的——生锈的电路板和单片机，复杂无解的电路，就像他。再之后有了苔原，小刘已经很久没发小说了。我还记得我在大一军训的某个早饭时间，打开苔原看了一篇小刘的小说，之后发动态说，看完之后感觉一天都被毁了。

    当然今天并不是要说小刘，我不愿意将他神化，甚至巴不得他是个废物——只有这样能让我感到安心。我要讨论的是卡佛——雷门特·卡佛。

    他太像小刘了，或者说，小刘太像他了。我一直认为世界上有两种人：一种有希望，一种没有。前者的特点很好概括：他们热爱生活，不畏困难，将挫折视为人生常态，一旦有了目标就会努力去做；而后者则难以归类：他们有各种不幸用以招致今日的局面。

    而卡佛的文字只是加速了我看清自己的过程。拿到书后花了一个下午看完了。之后从教学楼走出来，下午6点，天还亮着。我只感到世界都不真实了，我走得很慢，步子很小，我甚至第一次发现我可以走得这么慢。

    我的生活是从什么时候开始变坏的？以前我觉得是初三之后，所有能让我感觉欣喜的记忆都停留在了15岁。之后遇到的人或事都无法让我体会到真正的快乐了（除去那段喜欢ice的时间，不过现在也已经失去了）。正如卡佛在The Third Thing That Killed My Father Off 里所说的一样。

    “It seemed to me everything took a bad turn for my father after that. Just like Dummy, he wasn’t the same man anymore. That arm coming up and going back in the water, it was like so long to good times and hello to bad.”

    但现在我觉得我大一大二的生活还是没有现在这么坏。那时候好像还有勇气去学习线性代数，还有勇气去面试，去认识新的人，参加运动会。但20岁后的我渐渐发现，我永远成为不了“那种人”——他们永远深深扎根，充满自信，向上昂扬生长。在他们心里自己是无所不能的，他们不那么在意别人的目光。他们也会觉得自己很弱，但却不会放弃。我永远成为不了那种人——他们认为无论如何生活是有希望的。

    他们竟会这样认为——而这是我的真实想法。我不知道我是怎么变成这样的。我无法怪罪任何人，因为这并非他们的过错；但我也没有做任何错事，无论多么客观，自己也无法评判自己。所以读小刘的文字和卡佛的文字都带给我异样的感觉：他们的小说都像发生在一个破败的三线城市中，黄昏时分，但是没有美丽的夕阳，只有昏暗——也没有张灯结彩和灯红酒绿。

    就像《你们为什么不跳舞》里的那个时刻，昏暗的傍晚，男人坐在门窗洞开的空房屋中，注视着眼前没有插电的电视，听着门外那对情侣的交谈。他们不知道男人遭遇了什么，似乎有所体会，却又无法言说。他们之间有层无形的屏障。

    独处时才能学着与自己相处，可我甚至不愿留给自己这样的时间。就像我做的每一件事都令自己厌恶，却无法摆脱。

    所以我觉得卡佛令我着迷是必然的。我能体会到废物的desperation，世上废物很多，但能将废物&绝望描写出来的人很少。所以我也不愿意看知乎精英们讨论卡佛成就的原因，在讨论这样一位作家时还不忘炫耀自己国际学生的身份，令人恶心。他们永远不会理解“每况愈下”的真实含义，他们的生活中不会出现大片的空白，他们也永远不会经历荒芜。

    所以我在想，最近总把“太难了”挂在嘴边可能也是一种纾解。好像我说出来了就能引发别人的共鸣，好让自己有一种“原来其他人都和我一样啊，放心了。”但说了一千一万次也好，收到了无数相似的回应也好，我最终确定的事实仍是，没人和我有同样的处境。连我都觉得自己荒谬——缺乏毅力，不愿思考，却仍渴望着做一个特殊的人，仍期盼着会有人走过来拉我一把。

    而卡佛的解答是，在真实生活的场景下，除了自救，没人能够帮上任何实质性的忙。

    这也是他所有小说的最终主题吧。谁也帮不了你，除了你自己。

  • 你真的想回到过去吗？

    2019-03-27 /
    • { 随笔 }

    今天买的火影护额引起我不少感触，它让我想起小时候经过门口小商店时内心复杂的感受：想看，想买，但是没有钱，就算有钱，若是被家长发现自己买了这些没用的东西，也会招致斥责。我一直都这么觉得：其实当未成年小孩子真没那么好。

    小学的时候陕4每天下午放学时间，会播一部陀螺战斗类的电视剧（其实只是卖陀螺的厂家拍的宣传用片，类似现在的软广，不过那时小孩子们都很吃这一套）。我那时特别喜欢。攒了10块钱（2010年左右）去买了个周边——一个黄色的项链，代表着雷电。项链的质感很不错。

    戴上后的第一天，我理所应当去学校炫耀。班里一个男生非要拽过去看，一拉一耸的就把链子拽断了。

    我当时觉得天都塌了，但面对同学也说不出什么。只能说，我这是十块钱买的，也不跟你加钱了，你就还我10块。

    催了一学期也没要来。我只能寄希望于学期末的家长会。那时小孩子心里的大人就是正义的，也不会考虑立场这回事。

    家长会结束后，义愤填膺地跑去那个男生的妈妈面前，说了这个事。在我脑海中反复上演了一学期的——他妈妈大发雷霆回家把他暴打一顿，再带着他来给我道歉——的戏码，没有发生。他妈妈反应淡淡的，说知道了。转过天，他没给我钱，给了我一条和原来那条不一样造型的项链，是另一个人物的。

    我心里并不满意这个结果，可也没什么办法。最后仍是不了了之。

    说了这么多，很杂乱，但并不是停留在表面上的“看到了火影回忆起了童年”。其实有很多更深层的东西。我有时候感觉我在情感方面很冷酷，人类社会中上到八十岁老人下到十几岁的小孩，大家都会不时回忆以前的生活，伴随发出想要回到过去的感慨。可我很少有这种经历。王姐cx总说想回到高中，而我和朱麒臻能在coco人手一杯茉莉绿茶从下午坐到晚上一起回忆初中，笑得坐不起身，晚上十点了还趴在吧台上不走。

    但我的怀念也就止步于此。小学意味着奥数的折磨与全方位的不自由的管制，而高中除了能够吞噬一切的自卑以外就是考试成绩，再无它物。高三时看到很多激励性质的语言，听的最多的无非那句“高考完的考生以为自己解放了，实际上他们刚刚离开了天堂”。我从那时就对这句话嗤之以鼻——即使高中有过一星半点的美好回忆，我也坚信，这些细碎的东西，永远无法与它带给我的痛苦相匹配。我那时就明白，即使我高中毕业，走入大学，步入社会，也绝不会存在那样的时刻——我怀念我的高中，想要再次投身到那个冰冷而强硬的大环境中去。

    所以说，长大真好。除了情感丰富之外什么也没有的日子，本质上仍是贫乏的。小学时的你不愿和所谓的朋友分别，高中时的你却已经不会再回首。随着长大，你的负担会越来越重，但也托了成长的福，你摆脱了很多束缚。 就算十年过去了，我还是很厌恶小学时路过门口小商店时那种源自内心的不自由感，不论其初衷是否合理。

    其实你没那么怀念过去。别脱口而出，别理所当然。

    CQL特性

    No arbitrary WHERE clause

    创建 create

    之后创建monkeys的表。

    CREATE TABLE monkeys (

    ​ type text,

    ​ PRIMARY KEY ((type, family), common_name));

    注意此处的primary key。此处为一个复合键。前一个(type, family)为partition key，后一个common_name为cluster key。

    composite partition key

    由多列组成，在cluster columns之前由一组额外的括号定义。

    既然cluster key决定了数据在一行中存储的顺序 那么partition key就帮助将数据分散在cluster中。

    设计表时，可将partition key视为将数据平均地传播到cluster中的工具。可将cluster key视为决定一个partition中数据顺序的事物。

    *cluster key是可选的。

    插入 insert

    当插入数据时，主键是必需的（mandatory）。

    命令语句同MySQL数据库。

    之后用select语句查看数据，红色属性为partition key，蓝绿色（cyan）属性为cluster key。

    一致性等级 consistency level

    此属性可以配置。

    Cassandra作为NoSQL数据库在CAP原理上选择了AP，即availability和partition tolerance（未选择C which means consistency）。

    输入 consistency cql中出现表述：

    cqlsh:animals> consistency
    Current consistency level is ONE.

    解释：一致性等级为1，表示只要一个node成功插入了这个statement，则认为插入成功。在每一个节点上，该insert statement首先被写入commit log，之后再被插入memtable（如一个write back cache）。这个memtable会被周期性或是在达到某个

    bu’guo阈值时被冲入disk中。这个flushing的过程将memtable/cache转化为一个SSTable。

    节点工具 flush

    手动将animals flush进入disk以创建SSTable

    nodetool flush animals

  • 关于爱与偏爱

    2018-03-05 |
    • { 荆棘鸟 }
    /
    • { 书评 }

    开学第二周的周一下午，两节E类课，把《荆棘鸟》最后的五十来页看完了。可能是因为坐在几十个人之间看完了结局，结局没让我哭。但是吃饭的时候在豆瓣上看到这段话还是没忍住：

    “三十七年，拉尔夫和梅吉只有五次相见：1932年，梅吉的父亲和哥哥斯图尔特在德罗海达的大火中死去，拉尔夫在泥泞之中艰难跋涉，狂奔而来；1937年，梅吉将要生下卢克的女儿朱丝婷，拉尔夫在床前握住她的手；1938年，拉尔夫与梅吉在麦特劳克岛，跨越了身为一名教士最后的底线；1952年，拉尔夫受封红衣主教，回到德罗海达，见到14岁的戴恩，却不知那正是他和梅吉的儿子；1965年，戴恩死了，得知真相的拉尔夫悲痛欲绝，亲自为戴恩做完追思弥撒之后，死在梅吉怀中，长眠在德罗海达。”

    我在想，从这本书进入主线剧情之后，支撑我继续看下去的就是梅吉和拉尔夫这两人的情感了。其实心里也清楚，拉尔夫不会放弃自己的信仰与事业，但还是有所奢望，想着深沉的情感会产生奇迹。可是没有什么奇迹。对两人的结局不满意，可是也想不到在这种关系下什么样的结果才能让所有人都称心。梅吉在11岁时便已经沉浸在拉尔夫的宠溺中了，在那个盛大的集会上，拉尔夫抱着她，踏过水坑，却故意丢下那个美丽而会骑马的姑娘去提着笨重的裙子趟过脏水。这就已经足够令人动容了，我们想要的不仅仅是爱，如果是偏爱才更好。小小的梅吉那时就已经知道自己对于拉尔夫是与其他女孩子不同的，被自己热爱崇敬的人所偏爱，多么令人快乐的事啊。

    所以他们算是幸运了。感受过双方深沉的爱：梅吉在拉尔夫的万般疼爱中长大，而拉尔夫知道了梅吉处在分娩的痛苦中时，口中喊着自己的名字。而且在那个与世隔绝的小岛上，他们有了孩子。想想也就够了，即使他们最终没有结婚，她最终也不是梅吉·布里克萨特。
