  • 《流浪地球》观后感

    2019-02-06 |
    • { 电影 }
    • { 流浪地球 }
    /
    • { 影评 }

    这部电影证明我国电影特效技术早都到位了 只是一直没遇到好作品

    几十年后 严酷的地上环境 人类为了生存躲进避难所 辐射里称为shelter 这里称为地下城 说实话爷爷抱着孙子进入避难所的这幕过场给我留下了很深的印象 简直和辐射4开场时一模一样 而教室外阳光明媚的虚拟布景又让我想起海伯利安的模拟刺激接入 总而言之就是要素过多 而又恰到好处

    我总觉得悲伤和壮烈 这两点中的任何一点 被单独提及时都无法震撼人心 只有当它们以悲壮出现时才是最优解：“虽千万人 吾往矣” 我想不论何时 科幻这一题材终究是载体 它的内核仍是要落到人类的情感之上 所以我才会在看到刘启一人抬头仰望巨大奇异的木星时眼眶湿润 被迫与不可控的宇宙星体相对抗 像极了温伯特教授带着逆时间而行的瑞秋回到海伯利安寻找答案 他们都面临着不可知的命运 企图以一己之力扭转早已落幕的结局 这是悲伤 而当刘启在绝望的气氛中寻找解答与出口 当温伯特跪倒在浑身布满荆棘的伯劳面前时 他们确实发出了有力的声音 为了抓住仅存的微光 这是壮烈

    科学仍然很迷人 它为你描绘出无限大的蓝图 又伴随着深不可测的冷酷与温情 无法预测 无法改变 也无法与之抗衡

  • 《海伯利安》阅读记录

    2018-01-15 |
    • { 科幻 }
    • { 海伯利安 }
    /
    • { 书评 }

    First story 神父

    保罗·杜雷神父

    为了拯救教会编造谎言而被放逐

    追逐毕库拉 ——海伯利安上的土著人

    他见到了伯劳

    他最终也成为了十字架的一员

    为了摆脱十字架 他将自己钉在巨树中

    十字架的人无法命享真死

    他在苏醒-痛苦-死亡中轮回了七年

    十字架一遍遍将他唤醒 强迫他承受巨大的痛苦

    “我无论如何也要回去 可他如何……七年啊！”

    霍伊特神父终结了保罗神父的痛苦 使他得到了解脱

    霍伊特神父接替并承载了两个十字架的痛苦

    Second story 上校

    卡萨德上校

    在虚拟世界里的神秘女人

    与他一次次相遇 带他逃脱危险

    大哀之君是伯劳

    伯劳为他赋予停止时间的能力

    她逆时间而来 他迎时间而上

    未来的荆棘之树上 至少悬挂着七个人中的某个人

    Third story 诗人

    马丁·塞利纳斯

    从旧地而来 由于运气而卖出三十亿的诗集

    人生大起大落

    去往海伯利安

    哀王 比利王

    他的女神是伯劳

    只有伯劳不断杀戮 他的史诗才能继续下去

    最终只剩下他与伯劳存在于此

    伯劳杀死了想要毁掉手稿的比利王 马丁点燃了圣母圣子像

    他的诗集最终没有完成

    Forth story 学者

    索尔·温伯特

    看着孩子逆时间而去

    每一个清晨的迷茫与痛苦

    都成为了父母的煎熬

    生命的最初意味着结束

    “从未来回到过去 时光不可追溯”

    誓与伯劳对抗 反抗不可知的命运

    Fifth story 侦探

    拉米亚·布劳恩

    深陷于赛博人的目光

    他是约翰·济慈吗

    他是谁

    但他的生命被奉献给了教会

    连同他眼中晶莹剔透的光芒一起

    在主教面前抱着那具本就没有生命的尸体

    海伯利安能给她答案吗

    Sixth story 领事

    希莉
