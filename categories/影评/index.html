  • all胜我真的可以

    2019-03-15 |
    • { 我的英雄学院 }
    • { 爆豪胜己 }
    /
    • { 影评 }

    期待了一个月的小英雄剧场版，今天下午去电影院阅毕，流泪走出电影院。

    其实全程内心都有一个声音在干嚎：

    欧叔和David俩人的关系是不是有点过于那个了

    欧叔年轻时候也太帅了 我晕

    轰也太苏了吧怎么会有这种人啊

    爆豪和谁一起都好吃

    咔酱带了切切一起出来玩 切爆 is real

    咔酱和轰总也太般配了 兄弟们觉得般配的能把般配打在公屏上吗

    咔酱为什么见到deku就开始生气啊 是不是有点那个

    到了最后：all胜 我真的可以！

    细节太多了以致于现在我语无伦次，虽然剧场版二人交流不多，总结下来内心对于出胜这对还是意难平。前段时间看完第三季的倒数第二集时，我写下了这段话：

    语言无法描述这种羁绊。他们热切地想要靠近彼此，想要紧紧拥抱对方，但又只能以互相伤害的方式相处。心里说着“别过来”却又不可遏止地步步紧逼，渴望着平等与交流。但是自尊不允许他这样。两人都说过“明明以前都是你在我身后”，另一方也都曾撕心裂肺地喊出过「返せ」。

    他们绝不可能在一起，绝不会成为正常环境下的友好伙伴，但对于彼此来说永远是最重要的存在。永远无法接近，永远伤痕累累。

    永远爱恨交加。

    再联系那篇《漆黑一片》，有时候就觉得人的情感真的太微妙了。现实的情境下人的所有情感往往是处于dilemma中的，进退不得，像是投鼠忌器，或是前有狼后有虎，但也正因此才达到了微妙的平衡，才会让人觉得“这才是真实的他们啊”。

    比如出胜，比如佐鸣/鸣佐（这对怎么样我都可以）。

    想到什么以后再加上去，二刷希望。

  • 《流浪地球》观后感

    2019-02-06 |
    • { 电影 }
    • { 流浪地球 }
    /
    • { 影评 }

    这部电影证明我国电影特效技术早都到位了 只是一直没遇到好作品

    几十年后 严酷的地上环境 人类为了生存躲进避难所 辐射里称为shelter 这里称为地下城 说实话爷爷抱着孙子进入避难所的这幕过场给我留下了很深的印象 简直和辐射4开场时一模一样 而教室外阳光明媚的虚拟布景又让我想起海伯利安的模拟刺激接入 总而言之就是要素过多 而又恰到好处

    我总觉得悲伤和壮烈 这两点中的任何一点 被单独提及时都无法震撼人心 只有当它们以悲壮出现时才是最优解：“虽千万人 吾往矣” 我想不论何时 科幻这一题材终究是载体 它的内核仍是要落到人类的情感之上 所以我才会在看到刘启一人抬头仰望巨大奇异的木星时眼眶湿润 被迫与不可控的宇宙星体相对抗 像极了温伯特教授带着逆时间而行的瑞秋回到海伯利安寻找答案 他们都面临着不可知的命运 企图以一己之力扭转早已落幕的结局 这是悲伤 而当刘启在绝望的气氛中寻找解答与出口 当温伯特跪倒在浑身布满荆棘的伯劳面前时 他们确实发出了有力的声音 为了抓住仅存的微光 这是壮烈

    科学仍然很迷人 它为你描绘出无限大的蓝图 又伴随着深不可测的冷酷与温情 无法预测 无法改变 也无法与之抗衡
