  • 你是真的想回到过去吗？

    2019-03-27 /
    • { 随笔 }

    今天买的火影护额引起我不少感触，它让我想起小时候经过门口小商店时内心复杂的感受：想看，想买，但是没有钱，就算有钱，若是被家长发现自己买了这些没用的东西，也会招致斥责。我一直都这么觉得：其实当未成年小孩子真没那么好。

    小学的时候陕4每天下午放学时间，会播一部陀螺战斗类的电视剧（其实只是卖陀螺的厂家拍的宣传用片，类似现在的软广，不过那时小孩子们都很吃这一套）。我那时特别喜欢。攒了10块钱（2010年左右）去买了个周边——一个黄色的项链，代表着雷电。项链的质感很不错。

    戴上后的第一天，我理所应当去学校炫耀。班里一个男生非要拽过去看，一拉一耸的就把链子拽断了。

    我当时觉得天都塌了，但面对同学也说不出什么。只能说，我这是十块钱买的，也不跟你加钱了，你就还我10块。

    催了一学期也没要来。我只能寄希望于学期末的家长会。那时小孩子心里的大人就是正义的，也不会考虑立场这回事。

    家长会结束后，义愤填膺地跑去那个男生的妈妈面前，说了这个事。在我脑海中反复上演了一学期的——他妈妈大发雷霆回家把他暴打一顿，再带着他来给我道歉——的戏码，没有发生。他妈妈反应淡淡的，说知道了。转过天，他没给我钱，给了我一条和原来那条不一样造型的项链，是另一个人物的。

    我心里并不满意这个结果，可也没什么办法。最后仍是不了了之。

    说了这么多，很杂乱，但并不是停留在表面上的“看到了火影回忆起了同年”。其实有很多更深层的东西。我有时候感觉我在情感方面很冷酷，人类社会中上到八十岁老人下到十几岁的小孩，大家都会不时回忆以前的生活，伴随发出想要回到过去的感慨。可我很少有这种经历。王姐cx总说想回到高中，而我和朱麒臻能在coco人手一杯茉莉绿茶从下午坐到晚上一起回忆初中，笑得坐不起身，晚上十点了还趴在吧台上不走。

    但我的怀念也就止步于此。小学意味着奥数的折磨与全方位的不自由的管制，而高中除了能够吞噬一切的自卑以外就是考试成绩，再无它物。高三时看到很多激励性质的语言，听的最多的无非那句“高考完的考生以为自己解放了，实际上他们刚刚离开了天堂”。我从那时就对这句话嗤之以鼻——即使高中有过一星半点的美好回忆，我也坚信，这些细碎的东西，永远无法与它带给我的痛苦相匹配。我那时就明白，即使我高中毕业，走入大学，步入社会，也绝不会存在那样的时刻——我开始怀念我的高中，想要再次投身到那个冰冷强硬的大环境中去。

    所以说，长大真好。除了情感丰富之外什么也没有的日子，本质上仍是贫乏。小学时的你不愿和所谓的朋友分别，高中时的你却已经不会再回首。随着长大，你的负担会越来越重，但也托了成长的福，你摆脱了很多束缚。

    至少，在回到过去这件事上，别脱口而出，别理所当然。
