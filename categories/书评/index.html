    First story 神父

    保罗·杜雷神父

    为了拯救教会编造谎言而被放逐

    追逐毕库拉 ——海伯利安上的土著人

    他见到了伯劳

    他最终也成为了十字架的一员

    为了摆脱十字架 他将自己钉在巨树中

    十字架的人无法命享真死

    他在苏醒-痛苦-死亡中轮回了七年

    十字架一遍遍将他唤醒 强迫他承受巨大的痛苦

    “我无论如何也要回去 可他如何……七年啊！”

    霍伊特神父终结了保罗神父的痛苦 使他得到了解脱

    霍伊特神父接替并承载了两个十字架的痛苦

    Second story 上校

    卡萨德上校

    在虚拟世界里的神秘女人

    与他一次次相遇 带他逃脱危险

    大哀之君是伯劳

    伯劳为他赋予停止时间的能力

    她逆时间而来 他迎时间而上

    未来的荆棘之树上 至少悬挂着七个人中的某个人

    Third story 诗人

    马丁·塞利纳斯

    从旧地而来 由于运气而卖出三十亿的诗集

    人生大起大落

    去往海伯利安

    哀王 比利王

    他的女神是伯劳

    只有伯劳不断杀戮 他的史诗才能继续下去

    最终只剩下他与伯劳存在于此

    伯劳杀死了想要毁掉手稿的比利王 马丁点燃了圣母圣子像

    他的诗集最终没有完成

    Forth story 学者

    索尔·温伯特

    看着孩子逆时间而去

    每一个清晨的迷茫与痛苦

    都成为了父母的煎熬

    生命的最初意味着结束

    “从未来回到过去 时光不可追溯”

    誓与伯劳对抗 反抗不可知的命运

    Fifth story 侦探

    拉米亚·布劳恩

    深陷于赛博人的目光

    他是约翰·济慈吗

    他是谁

    但他的生命被奉献给了教会

    连同他眼中晶莹剔透的光芒一起

    在主教面前抱着那具本就没有生命的尸体

    海伯利安能给她答案吗

  • 关于爱与偏爱

    2018-03-05 |
    • { 荆棘鸟 }
    /
    • { 书评 }

    开学第二周的周一下午，两节E类课，把《荆棘鸟》最后的五十来页看完了。可能是因为坐在几十个人之间看完了结局，结局没让我哭。但是吃饭的时候在豆瓣上看到这段话还是没忍住：

    “三十七年，拉尔夫和梅吉只有五次相见：1932年，梅吉的父亲和哥哥斯图尔特在德罗海达的大火中死去，拉尔夫在泥泞之中艰难跋涉，狂奔而来；1937年，梅吉将要生下卢克的女儿朱丝婷，拉尔夫在床前握住她的手；1938年，拉尔夫与梅吉在麦特劳克岛，跨越了身为一名教士最后的底线；1952年，拉尔夫受封红衣主教，回到德罗海达，见到14岁的戴恩，却不知那正是他和梅吉的儿子；1965年，戴恩死了，得知真相的拉尔夫悲痛欲绝，亲自为戴恩做完追思弥撒之后，死在梅吉怀中，长眠在德罗海达。”

    我在想，从这本书进入主线剧情之后，支撑我继续看下去的就是梅吉和拉尔夫这两人的情感了。其实心里也清楚，拉尔夫不会放弃自己的信仰与事业，但还是有所奢望，想着深沉的情感会产生奇迹。可是没有什么奇迹。对两人的结局不满意，可是也想不到在这种关系下什么样的结果才能让所有人都称心。梅吉在11岁时便已经沉浸在拉尔夫的宠溺中了，在那个盛大的集会上，拉尔夫抱着她，踏过水坑，却故意丢下那个美丽而会骑马的姑娘去提着笨重的裙子趟过脏水。这就已经足够令人动容了，我们想要的不仅仅是爱，如果是偏爱才更好。小小的梅吉那时就已经知道自己对于拉尔夫是与其他女孩子不同的，被自己热爱崇敬的人所偏爱，多么令人快乐的事啊。

    所以他们算是幸运了。感受过双方深沉的爱：梅吉在拉尔夫的万般疼爱中长大，而拉尔夫知道了梅吉处在分娩的痛苦中时，口中喊着自己的名字。而且在那个与世隔绝的小岛上，他们有了孩子。想想也就够了，即使他们最终没有结婚，她最终也不是梅吉·布里克萨特。

  • “我的天空里没有太阳”

    2016-11-25 |
    • { 白夜行 }
    /
    • { 书评 }

    下午三节线代听的脑子昏昏沉沉的，下了课决定去买两天没吃的面包和饼干，然后去软件楼把《白夜行》剩下的两章看完。

    大概看到六点多吧，吃着饼干看完的。太突兀了，我甚至不敢相信自己看到了封底的书籍信息。亮司毅然决然地跳下去了，雪穗转身离开，“保持了最大程度的克制”，是这样吧？

    “亮司为了雪穗一辈子没有出错，最后雪穗怎么能出错呢？”

    下午看到这句话就觉得心里很难受，现在深夜坐在电脑前，还是没能忍住眼泪。不论两人做了多少的恶事，但实际上从一开始，他们就是坏蛋大人的牺牲品。雪穗被亲生母亲卖给了恋童癖老板，而亮司作为雪穗最亲密的朋友，目睹了丑恶的一幕。从他弑父的那一刻开始，两人就注定了要一直互利共生下去。

    在看书的过程中，我试图去想象并构建背后的故事。两人于何时邂逅，于何时确定心意，于何时变得密不可分……他们在书中甚至没有一句对话。可我想对双方来说，两人在一起的时候都是最放松的时刻，不需要再提防可能来自外界的猜测或恶意。两人因“恶”而被连结一起，亮司说自己“在白夜中行走”，雪穗说“我的天空里没有太阳”“因为有东西代替了太阳”“我从来就没有太阳，所以不怕失去。”

    其实我一直很讨厌自己长久而持续地处于和另一个唯一的人相互依赖的立场上，比如来到新环境后立刻去搜寻一个能和自己一起吃饭一起占座位一起去参加活动的女生，也许是因为心里总存有芥蒂，连吃饭也要刻意地避开同学，能少说一句便少说一句。因为处于两人小组中总让我有莫名无力的感觉，知道自己是个不可依靠的人，也很难放下心来去完全地依靠别人。我不知亮司和雪穗在杀死桐原老板后会不会有这样的感觉，可最终他们能够完全地依靠对方。亮司能为雪穗处理一切麻烦事，从初中时散布负面谣言的女同学，到二婚后敏感而抵触的继女，十九年，从始至终，两人都行走在白夜中。亮司的人生没有白天，雪穗的天空没有太阳，那么他便是她的太阳，她是他白色的影子。他们是只为对方开启的光源，是光，是雪。

    我还是讨厌两个人的组合，不论是谁，是怎样的人。因为，一旦有一个人离开，这样的组合不就消失了吗？那么，对于剩下的那只鸳鸯，不论雌雄，不是太过残忍了吗？现在想来，雪穗并不是克制，她的动摇已经外化了。若要装作不认识，只应表现出惊讶恐惧即可，但她转身离开，留下白色的影子。我讨厌这种失去最后的依靠的感觉。就像军训的那天早上，就已经有不好的预感。我还记得自己当时是怎么想的，“最后一根稻草也失去了”。那么，像刘书宇说的那样，让人变得很想跳湖。

    他们利用过很多人，也害死了很多人，这毋庸置疑。而同时，加害者在最初也是受害者。所以这是一个以怨报怨的故事，并不高尚。

    只是我一开始没想到雪穗对继女美佳说的话，竟然是真的。

    我不想为作恶者辩护，人类社会发展至此，恶是找不到根源的。而二人也只是选择用自己曾经被世界对待的方式报复这个世界而已。理性方面，儿童时期埋下恶的种子，这种子必会开出有毒的花，结出有毒的果。感性方面，在白夜中互相照明的二人，在黑暗中扶持前行的二人，足以令人动容。但更让我坚信二人组是不可靠的。

    很多女生都习惯性地觉得爱情比友情更重要，而他们二人的情感甚至比爱情更牢固——依赖。我只能想到这个词，是能为了对方毁掉或获得一切的勇气，是能放心向后坠落也不怕着地的信任。

    总结，是个悲剧，但雪穗在最后也没有做错，至少她有过这样的经历，有过这样幸福的日子，有过彼此。
